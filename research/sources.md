# Sources

## Verified sources

- [MLX repository](https://github.com/ml-explore/mlx)  
  Confirms MLX is an array framework for Apple silicon, includes Python/C++/Swift APIs, lazy computation, dynamic graphs, and unified memory.

- [Qwen3 repository](https://github.com/QwenLM/Qwen3)  
  Confirms the current Qwen3 family, dense and MoE variants, local-running guidance, and the availability of dense sizes such as 32B.

- [autoskills repository](https://github.com/midudev/autoskills)  
  Confirms stack detection, automatic skill installation, and generated `CLAUDE.md` support when targeting Claude Code.

- [llama.cpp repository](https://github.com/ggml-org/llama.cpp)  
  Confirms a mature local inference runtime and serving ecosystem that can be used as a fallback or compatibility path.

- [gnhf repository](https://github.com/kunchenguid/gnhf)  
  Confirms a long-running agent loop that commits each successful iteration, rolls back failed ones, supports worktrees, and is explicitly designed for autonomous overnight execution.
